digi2 Essentials v1.0

Component library for Webflow. One script tag, modular, on-demand loading.

Installation

Single script tag in Webflow → Site Settings → Custom Code → Head Code:

<!-- Load only what you need via d2-* attributes -->
<script
  src="https://cdn.jsdelivr.net/gh/Digi2-Agency/digi2-essentials@latest/dist/digi2-loader.min.js"
  d2-gtm="GTM-XXXXXXX"
  d2-popups
  d2-cookies
  d2-forms
></script>
AttributeModuleSize (min)
d2-gtm="ID"Consent Mode V2 + GTM2.5 KB
d2-popupsPopup / modal manager8.4 KB
d2-cookiesCookie helpers1.1 KB
d2-formsForms + validation + UTM12.1 KB

Loader itself: 3.4 KB. Only requested modules are fetched.

Add d2-debug-mode to the loader tag:

<script src="...digi2-loader.min.js" d2-popups d2-debug-mode></script>

Forms

Wrap form in <div data-d2-form="name">. Auto-injects hidden tracking fields + validation.

digi2.forms.create('contact', {
  ipTracking: true,
  inputOnError: {
    borderColor: '#ef4444',
    boxShadow: '0 0 0 2px rgba(239,68,68,0.2)',
  },
  inputOnValid: {
    borderColor: '',
    boxShadow: '',
  },
})

Standalone Validation

digi2.forms.validate('hello@test.com', { required: true, email: true })
// → { valid: true, errors: [] }

digi2.forms.validate('', { required: true })
// → { valid: false, errors: ['required'] }

Custom Rules

digi2.forms.addRule('noBadWords', function(value) {
  return !['spam', 'test'].some(w => value.includes(w))
})

Google / Consent

Handles Consent Mode V2, GTM injection, consent manager. Configured via d2-gtm="GTM-XXX" on the loader tag.

Consent Categories

API

digi2.google.consent.get()
digi2.google.consent.grant('analytics_storage')
digi2.google.consent.deny('ad_storage')
digi2.google.consent.update({ analytics_storage: 'granted' })
digi2.google.consent.grantAll()
digi2.google.consent.denyAll()
digi2.google.consent.reset()
digi2.google.consent.categories()
digi2.google.dataLayerPush({ event: 'custom' })
digi2.google.getGtmId()

Events

digi2.on('loaded', fn)              // all modules loaded
digi2.on('module:loaded', fn)       // single module (receives name)
digi2.on('module:error', fn)        // module failed (receives name)
digi2.on('consent:updated', fn)     // consent changed (receives state)
digi2.off('loaded', fn)             // remove listener
digi2.off('loaded')                // remove all for event
digi2.emit('custom', data)         // emit custom event
digi2.onReady(fn)                 // alias for on('loaded')

Subscribing to loaded after modules already loaded calls fn immediately.

Module Management

digi2.modules.check('cookies')                   // true / false
digi2.modules.list()                              // ['popups', 'cookies']
await digi2.modules.require('forms')              // load on demand
await digi2.modules.requireAll(['cookies', 'forms'])
